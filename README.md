# Blinkit Retail Sales Dashboard 📊

## Overview
Interactive Power BI dashboard analyzing 8,523 retail items across outlet types and regions.

## Key Results
- Total sales tracked: $1.20M
- Tier 3 outlets: $472K (40% of sales)
- Supermarket Type1: $787K revenue
